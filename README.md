# 🎵 Music Catalog
 
> Your personal music library — browse, preview, and organize your collection.
 

## 📖 About
 
Music Catalog is a web application that lets you manage and explore your music collection from your browser. Add songs, preview demos, filter by genre or artist, link to streaming services, and build personalized playlists all in one place.
 
---
 
## ✨ Features
 
| Feature | Description |
|---|---|
| 🎵 **Song Management** | Add, edit, and delete songs from your catalog |
| 🎧 **Demo Previews** | Listen to a demo of any song before adding it to a playlist |
| 🔍 **Smart Filtering** | Filter your library by genre, artist, or release year |
| 🔗 **Streaming Links** | Link any song directly to Spotify or other streaming services |
| 📋 **Playlists** | Create and manage your own custom playlists |

## 🚀 Getting Started
 
### Prerequisites
 
- Python 3.8 or higher
- Git
- pip (Python package manager)
 
### Installation For Developers
 
**1. Clone the repository**
 
```bash
git clone https://github.com/muhammada06/Music-Catalog <folder-name>
cd <folder-name>
```
 
**2. Install dependencies**
 
```bash
pip install -r requirements.txt
```
 
**3. Run the application**
 
Open `run.py` and press **Run**, or via terminal:
 
```bash
python run.py
```
 
**4. Open in your browser**
 
```
http://127.0.0.1:8080/
```
